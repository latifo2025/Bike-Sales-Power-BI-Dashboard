# Bike-Sales-Power-BI-Dashboard
I created a Power BI dashboard analysing sales for a bike sales company.
It tracks KPIs, compares regional performances, analyses product-level trends, and identifies high-value customers for bike and accessory sales.

**Steps:**
1. Connecting & Shaping Data : 
   I built an automated workflow to extract, clean, transform, and load the data using Power Query.
2. Creating a Relational Data Model :
   Built a data model using a normalised star schema
3. Added calculated fields with DAX
4. Visualizing Data with Reports :
   Added interactivity with bookmarks, slicer panels, parameters, tooltips, and report navigation
